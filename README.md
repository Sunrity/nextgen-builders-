## 🌍 About The Project

**Next Generation Builders** is a mission-driven platform focused on raising a new generation of African leaders.

We empower young people through:
- 🧠 Mindset Development  
- 🏆 Leadership Training  
- 💻 Skill Acquisition  

This web platform serves as the central hub for our programs, mentorship, and opportunities.

---

## ✨ Features

- 📚 Skill scholarship application system  
- 👥 Mentorship and community support  
- 🌐 Informational pages (About, Programs, Blog)  
- 📱 Fully responsive modern UI  
- ⚡ Fast and optimized performance

## ⚙️ Getting Started (Local Development)

Run the project locally:

```bash
# 1. Clone the repository
git clone <YOUR_GIT_URL>

# 2. Navigate into the project
cd <YOUR_PROJECT_NAME>

# 3. Install dependencies
npm install

# 4. Start development server
npm run dev

src/
│
├── components/     → Reusable UI components (Navbar, Cards, etc.)
├── pages/          → Main pages (Home, About, Programs, etc.)
├── assets/         → Images, icons, and static files
├── styles/         → Custom CSS files
└── main.tsx        → App entry point
